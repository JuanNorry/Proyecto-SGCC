# Proyecto SGCC (Sistema de Gestión de Comedores Comunitarios)

Este proyecto forma parte del Seminario de Práctica Informática.  
Consiste en el desarrollo de un sistema en Java con base de datos MySQL  
para la gestión de movimientos, productos y usuarios en un comedor comunitario.

## Estructura
- `/src` → código fuente Java  
- `Main.java` → punto de inicio del programa  
- `.gitignore` → exclusión de archivos temporales y binarios  

## Tecnologías
- **Java 22**
- **MySQL 8**
- **Visual Studio Code**
- **GitHub Desktop**
